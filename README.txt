# MB Casino UI (Plantilla)

Estructura lista para subir a GitHub Pages. Diseño mobile-first inspirado en tu captura.

## Pasos rápidos
1. Sube esta carpeta al repo (por ej. `mb-casino`) y activa GitHub Pages (branch `main`, carpeta `/root`).
2. Edita `index.html` para nombre/branding y `css/styles.css` si querés cambiar colores.
3. Reemplaza imágenes en `assets/img/` (manteniendo los nombres) o agrega nuevas y actualiza `games.json`.
4. Cambia el teléfono de WhatsApp en `js/app.js` (`phone = "5491112345678"`).
5. Crea más páginas de proveedores duplicando archivos en `/proveedores/` y filtrando por `provider`.

¡Listo!
